clear_utils

A modular utilities library--made for common auxiliary problems. Easily set Dates, Cookies, and more coming soon.

Description

This is a lightweight, simple utility library that I created to solve some of my common time-sink aspects of writing javascript applications. It's a project I hope to be helpful to me and if it helps others...great. This repo is a work in progress, and hopefully I can add more modules at a reasonable rate.

Modules

  • Dates
  • Cookies
  • LocalStorage (coming soon)
  • SessionStorage (coming soon)
